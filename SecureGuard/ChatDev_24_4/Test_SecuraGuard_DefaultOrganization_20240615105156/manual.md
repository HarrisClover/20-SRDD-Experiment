# SecureGuard

SecureGuard is a security software application that provides real-time monitoring and protection for personal computers. It constantly scans for malware, viruses, and unauthorized access attempts, alerting the user and taking measures to neutralize threats. It includes a firewall and password manager to ensure data privacy and security.

## Quick Install

As SecureGuard is a C++ application, you need to have a C++ compiler installed on your system. You can install it using the following commands:

For Ubuntu:
`sudo apt-get install g++`

For Windows, you can download it from [MinGW-w64](https://mingw-w64.org/doku.php).

After installing the C++ compiler, you need to install the required libraries. You can do this by running the following command in the project directory:

`make install`

## Main Functions

**Real-Time Monitoring:** SecureGuard constantly monitors your system for malware, viruses, and unauthorized access attempts. If a threat is detected, it alerts the user.

**Threat Neutralization:** Upon detecting a threat, SecureGuard takes measures to neutralize it. This could involve deleting infected files, blocking unauthorized access, etc.

**Firewall:** SecureGuard includes a firewall to block unauthorized access to your system.

**Password Manager:** SecureGuard includes a password manager to help you manage your passwords and ensure data privacy and security.

## How to Use

After installing SecureGuard, you can start it by running the following command in the project directory:

`make run`

SecureGuard will start monitoring your system for threats. If a threat is detected, it will alert you and take measures to neutralize the threat.

For more detailed information on how to use SecureGuard, please refer to the user guide included in the project directory.
